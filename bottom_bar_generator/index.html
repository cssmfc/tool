About..

This is an image based overlay generator.

Generated graphic can be used with OBS Studio or SplitCam Software as image source overlay

After click on Generate button, right-click on generated image, Save As... in your pc/laptop

How to... options and buttons

Background - dropdown option from where different pre-set backgrounds can be selected

Slots Skin - dropdown option from where different social slot skins can be selected

Change Font - dropdown option from where different fonts can be selected

Font Style - dropdown option from where different font styles can be changed (bold, italic, slim, uppercase, no font transformation)

Font Color - dropdown option from where different colors can be applied to editable text

@username - type to change texts, add your usernames for websites

Apply button - updates all input texts with your own

x Hide button - toggle visibility for each social slot

Choose Icon - dropdown option from where different icons can be changed

Generate button - click to generate PNG image file

Refresh button - click refresh the page, all editing reverts back to default

No Icons button - click hide all icons
